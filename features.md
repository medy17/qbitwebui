# Feature Roadmap

## Themes & Styling

- [ ] Theme presets (Catppuccin, Dracula, Nord, Gruvbox, One Dark)
- [ ] Theme switcher with localStorage persistence
- [ ] Custom accent color picker
- [ ] Density modes (compact/comfortable/spacious)

## Functionality - High Value

- [ ] RSS feeds - manage subscriptions and auto-download rules
- [ ] Search - integrated torrent search across configured indexers
- [ ] Bandwidth scheduler - visual schedule for speed limits
- [ ] Sequential download toggle - per-torrent setting
- [ ] File priority management - skip/normal/high in content tab
- [ ] Labels/tags - beyond just categories
- [ ] Bulk operations - select multiple, batch actions

## Functionality - Quality of Life

- [ ] Notifications - browser notifications on complete
- [ ] Drag & drop - reorder queue, drop .torrent files
- [ ] Context menu - right-click on torrent row
- [ ] Torrent health indicator - availability percentage
- [ ] Copy magnet/hash - quick copy buttons
- [ ] Open destination folder - link to file manager

## Multi-instance

- [ ] Server switcher dropdown in header
- [ ] Save multiple qBittorrent endpoints
- [ ] Quick switch between home/seedbox/etc.

## Settings Panel

- [ ] Connection settings (ports, encryption)
- [ ] Speed limits (global/per-torrent)
- [ ] Download path defaults
- [ ] WebUI preferences
